HTML Basic Structure

Most HTML pages follow the same basic structure.

Important Parts

<!DOCTYPE html> tells the browser to use HTML5.

<html> is the main wrapper for the whole webpage.

<head> stores page information like the title.

<title> sets the browser tab name.

<body> contains the content users see.

Opening and Closing Tags

Most HTML elements have an opening tag and a closing tag.

Example: <p>This is a paragraph.</p>
