# HTML-CSS-Fundamentals

My 15-day HTML and CSS fundamentals learning journey with daily practice files.

## About This Repository

This repository contains beginner-friendly HTML and CSS practice projects completed over 15 days. Each day focuses on one important web development topic, starting from basic HTML and gradually moving into CSS styling, layout techniques, and a final landing page project.

The goal of this repository is to build a strong foundation in frontend web development using simple, readable, and practical examples.

## What I Learned

- Basic HTML document structure
- Headings, paragraphs, and text formatting
- Links and images
- Lists and tables
- HTML forms
- External CSS
- Colors and fonts
- Borders, margins, and padding
- CSS box model
- Display and position properties
- Backgrounds and gradients
- Flexbox and Grid layouts
- Building a complete landing page using HTML and CSS

## How to View the Projects

1. Open any day folder.
2. Open the `index.html` file in a browser.
3. For CSS days, make sure `style.css` is in the same folder as `index.html`.
