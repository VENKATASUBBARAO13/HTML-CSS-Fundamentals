HTML Introduction

What is HTML?

HTML is the standard language used to create the structure of webpages.

What does HTML stand for?

HTML stands for HyperText Markup Language.

Why do we use HTML?

HTML tells the browser what content should appear on a webpage, such as headings, paragraphs, links, images, lists, tables, and forms.

Beginner Tip

Think of HTML as the skeleton of a webpage. CSS is used later to make that skeleton look beautiful.
